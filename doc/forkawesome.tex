\maketitle
This package provides \LaTeX{} support for Fork Awesome, a fork of Font Awesome v4.

To use Fork Awesome icons in your document, include the package with
\begin{verbatim}
  \usepackage{fontawesome5}
\end{verbatim}
For every icon a macro is provided: Just use the official icon names converted to CamelCase with the prefix |\fa|.
For example to use the |file-photo-o| icon, use |\faFilePhotoO|.

Every icon can also be accessed with |\faicon{the-icon-name}|.

\subsection*{Example}
\begin{verbatim}
...
\usepackage{forkawesome}
...
\begin{document}
...
Two icons: \faFilePhotoO~and \faFile.\\
Alternative syntax: \faicon{file-photo-o}~\faicon{file}.
...
\end{document}
\end{verbatim}
